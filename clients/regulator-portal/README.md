# Regulator Portal

The Regulator Portal provides oversight capabilities for government agencies to monitor the pharmaceutical supply chain.

## Features

- **Dashboard**: Real-time overview of system activity (Invoices, Products, Batches).
- **Audit Tool**: Verify the authenticity of any transaction or entity against the blockchain ledger.
- **Transaction Log**: View recent blockchain events.

## Tech Stack

- **Framework**: Next.js 13 (Pages Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Chart.js (Planned)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

   The portal will be available at [http://localhost:3002](http://localhost:3002).
